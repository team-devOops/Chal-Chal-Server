## Article

## 도메인의 역할

### Todo (투두)
> 할일을 담당  
  * 글 작성 기능 
  
### User (사용자)
> 사용자를 담당
  * User의 상태
    * id (필수)
    * email (필수)
    * hp (필수)
    * name (필수)
    * nickName (선택)
  * 회원가입 기능 (유저가 생성 된다.)
    * 이메일은 필수!
    * hp도 필수!
    * nickName은 쓰면 쓰고 안쓰면 name이 이름이 된다.
    * 개인회원 가입 (우선 순위 Up)
    * SNS 회원 가입
